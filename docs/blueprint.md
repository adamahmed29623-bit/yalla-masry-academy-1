# **App Name**: Yalla Masry Academy

## Core Features:

- Firebase Project Creation: Allows users to create and manage Firebase projects.
- Service Configuration: Enable and configure Firebase services like Authentication, Firestore, Storage, Functions, etc.
- Rule Generation (AI): Generates security rules for Firestore and Storage using an AI tool, providing a baseline configuration and suggesting least-privilege permissions.
- User Authentication: Implement various authentication methods for user access.
- Database Management: Provide UI tools to visualize, query, and manage Firestore data.
- Real-time Monitoring: Dashboard for monitoring key metrics, usage, and performance of Firebase services.
- Deployment Pipeline: CI/CD pipeline configuration connected to code repositories to automatically deploy functions.

## Style Guidelines:

- Primary color: Dark slate blue (#37474F) to provide a professional and serious feel.
- Background color: Light grey (#ECEFF1), a very low saturation of the primary hue, providing good contrast.
- Accent color: Steel blue (#455A64), for calls to action.
- Body and headline font: 'Inter' (sans-serif) for a modern, clean, and easily readable interface.
- Use a set of clear, minimalist icons for services and actions.
- Clean, card-based layout to present Firebase services and configuration options in an organized manner.
- Subtle transitions and animations for a smooth user experience when navigating between sections and services.
